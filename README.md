# EstudioHub Site

Site institucional completo em Next.js para posicionamento comercial do EstudioHub.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Páginas

- `/` Home
- `/funcionalidades`
- `/planos`
- `/para-quem`
- `/integracoes`
- `/seguranca-governanca`
- `/sobre`
- `/contato`
- `/login`
